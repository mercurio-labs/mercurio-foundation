# Resources

`resources/` contains language-neutral foundation resources only.

- `foundation/empty.kir.json` is an empty, schema-versioned KIR document used by default path helpers.
- `foundation/representative.kir.json` is the shared, source-language-neutral KIR fixture used by distributed foundation tests.
- `foundation/core.rulepack.json` is a placeholder serialized rulepack path. The in-code core rulepack remains the authoritative default.

Versioned language resources belong in language-specific repositories.
