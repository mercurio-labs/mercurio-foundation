# Mercurio User Docs

This directory contains user-facing documentation for the public CLI and model workflows.

## Guides

- [CLI Guide](CLI.md): public command examples for parse, compile, lint, completions, and common input forms.
- [Project Descriptors](PROJECTS.md): `.mercurio-project.json`, provider kinds, and descriptor discovery.
- [KIR User Guide](KIR.md): compiled semantic JSON models and low-level workflows.
- [Querying And Evaluation](QUERY_EVALUATE.md): model queries, derived values, runtime context, and explanations.
- [KPAR Packages](KPAR.md): building and consuming `.kpar` model packages.
- [Troubleshooting](TROUBLESHOOTING.md): common command, descriptor, stdlib, KPAR, and Pilot-tool issues.

The [project README](../../README.md) gives the repository overview and quick start.

Development architecture and implementation plans live in [../development/README.md](../development/README.md).
